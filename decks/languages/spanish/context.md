# Spanish Area

Rioplatense Spanish (Argentine/River Plate dialect) Anki deck — vocabulary, grammar patterns,
and production cards targeting spoken Buenos Aires register.

## Files in this area

| Path | What |
|---|---|
| `./rioplatense-anki.md` | Main deck context: card types, generation rules, tagging taxonomy. Compiled via context-compiler before card generation. |
| `./focus_area.md` | Rioplatense dialect criteria (voseo, Argentine vocabulary). Injected into card generation and used as the sole extraction context for youtube-extract. |
| `./resources.md` | Lit-search guide: discovery sources, fetch sources, and author coverage table for Rioplatense reading material. |
| `./reading-log.md` | Log of reading sessions and material worked through. |
| `./reading-log/` | Reading-log area — mem-bank for reading session history. |
| `./files/` | Media files (PDFs, etc.) referenced by cards or resources. |
| `./backups/` | Deck backup archives (.apkg). |
