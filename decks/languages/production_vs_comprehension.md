# Production Cards vs Comprehension Cards (L1→L2 vs L2→L2)

**Date:** 2026-05-26  
**Question:** When learning vocabulary or expressions in a target language, should the retrieval cue be native-language (L1) or target-language (L2)? Is a target→target (L2→L2) card format more efficient than native→target (L1→L2) production at intermediate level?

Originated from Spanish deck idiom work but applies to any deck with a production goal.

---

## The Argument For L2→L2

- Content (videos, texts) often explains things *in the target language*, so the description is available
- Reading the L2 definition exposes you to related vocabulary "for free" (elaborative encoding)
- At intermediate level, thinking in L2 rather than translating through L1 is generally a good habit
- Richer encoding = stronger memory trace

## Why It Breaks Down in Practice

### 1. Retrieval cue mismatch — the core problem

Neither card format perfectly replicates real conversation — Anki is always a simplification. The question is: which simplification is *closer* to the retrieval path you'll actually use?

At early-to-intermediate level, the actual cognitive chain when producing a target-language expression is roughly:

1. Perceive situation
2. That situation maps to a concept already wired in native-language thought
3. From that concept, reach for the L2 form
4. Produce

The L1-cue card trains steps 2→3→4 directly — the path you are actually using. The L2-paraphrase card trains a sub-skill (L2 paraphrase → idiom form) that doesn't fit this chain at any stage and rarely fires in real speech.

Research confirms this for idioms specifically: L2 learners don't process idioms as lexicalized chunks the way natives do. Native speakers retrieve idioms faster than novel phrases because they're stored as whole units. Non-native speakers don't show that advantage — the idiom hasn't become a chunk yet. Getting there requires drilling the actual production trigger (situation → expression), not a proxy trigger (L2 paraphrase → expression).

**Caveat — level dependency.** At advanced level, L1-cue cards become a liability: they reinforce L1-mediated thinking instead of building direct situation→L2 wiring. At that point, neither card format is ideal — spontaneous production comes from real input and conversation, not from Anki. The level-appropriate format is:

| Level | Format | Reason |
|---|---|---|
| Beginner–intermediate | L1→L2 production | Matches the L1-mediated path you're actually using |
| Upper intermediate | L1→L2 with caution | Risk of reinforcing L1 crutch |
| Advanced | Real input/output | Cards can no longer replicate real situational triggers |

### 2. The "free vocabulary" argument is weak for production

Elaborative encoding is real — richer context does strengthen memory traces. But:
- Vocabulary encountered in the L2 definition is **read, not tested**. Retention of incidentally absorbed words is low without its own retrieval practice.
- If the definition contains unfamiliar words, cognitive load spikes and the target form encoding suffers — the card becomes about the definition words, not the item being learned.
- If the definition words are worth learning, make cards for them. Don't smuggle them as a side effect of another card.

### 3. Research on idiom and vocabulary production

Studies on L2 idiom acquisition (Liontas; Irujo) find that **context aids comprehension** of idioms but the effect on *production* is weaker. Production requires repeated successful retrieval from the cue you'll actually face. L1 transfer effects (when the target language has no structural equivalent) make production harder than comprehension; the card format needs to compensate by drilling the production direction explicitly.

Nation's research on vocabulary learning confirms that output-oriented retrieval practice — being forced to produce from a meaningful cue — yields stronger retention than input-only methods (reading definitions, even richly encoded ones).

## When L2→L2 Is Actually Useful

- **Passive reading/listening** — encountering material in the target language, not Anki
- **Comprehension-first cards** (rare, valid): when the goal is to recognize expressions in target-language reading, an L2→L2 cloze within a real sentence is appropriate
- **After you already produce the expression fluently** — deepening L2-mediated understanding at advanced level

---

## The Recommended Format

Keep the L2 explanation — but **move it to the back**, as supplementary context read *after* recall:

| Part | Content |
|---|---|
| **Front** | Situational native-language cue: *"Something's off here, this doesn't add up."* |
| **Back** | **Hay gato encerrado.** *(lit: there's a locked cat — something hidden or suspicious)* |
| **Back (context)** | *L2 definition from source, etymology, related vocab* |

This gives you:
- The correct retrieval path trained (situation → target expression)
- The rich encoding from the L2 explanation when you flip the card
- Related vocabulary exposed in context (passive benefit, no cognitive cost at recall time)

You don't lose the L2 content — it lives on the back. You just stop testing the wrong skill.
